# Развитие игровой индустрии за 2000–2013 гг.

## Описание проекта
Исследование развития игровой индустрии на основе открытых данных о продажах игр, платформах, жанрах и оценках пользователей и критиков за период с 2000 по 2013 год.

Цель — выявить ключевые тренды, определить самые популярные платформы и оценить распределение игр по качеству (оценкам пользователей и критиков).

---

## Задачи
1. **Сбор и предобработка данных**
   - Очистка данных от пропусков и дубликатов
   - Приведение названий столбцов к единому формату (snake case)
   - Преобразование типов данных
   - Заполнение пропусков в продажах (EU, JP) средними значениями по платформам

2. **Фильтрация данных**
   - Отбор игр, выпущенных в период с 2000 по 2013 год включительно

3. **Категоризация данных**
   - Разделение игр по оценкам пользователей:
     - Высокая (8–10)
     - Средняя (3–8)
     - Низкая (0–3)
   - Разделение игр по оценкам критиков:
     - Высокая (80–100)
     - Средняя (30–80)
     - Низкая (0–30)
   - Игры без оценок выделены в категорию "Без оценки"

4. **Анализ платформ**
   - Выделение топ-7 платформ по количеству выпущенных игр за период

5. **Формулирование выводов**
   - Обобщение результатов анализа
   - Выявление ключевых трендов игровой индустрии

---

## Стек
Python, Pandas, Jupyter Notebook

## Результаты
- Выявлены самые популярные платформы по количеству выпущенных игр
- Проведена категоризация игр по оценкам пользователей и критиков
- Обнаружено, что большинство игр получают средние оценки
- Плохих игр (с низкими оценками) выпускается значительно меньше, чем хороших и средних

## Общие выводы
- **Самая популярная категория оценок** — средняя (как у пользователей, так и у критиков)
- **Хороших игр выпускается значительно больше, чем плохих**
- **Лидеры среди платформ** — PS2 и DS (более 2 000 игр каждая)
