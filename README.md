# practicum_projects
Мои учебные проекты с курса Яндекс.Практикум по анализу данных.

## Список проектов

### 1. Анализ рынка общественного питания Москвы
[Подробнее](Moscow_restaurants_analysis/README.md)

**Задачи:**
1. **Сбор и предобработка данных**
   - Очистка данных от пропусков и выбросов
   - Приведение данных к единому формату
   - Обогащение данных (добавление признаков: режим работы, ценовой сегмент)

2. **Исследовательский анализ данных (EDA)**
   - Анализ структуры заведений по типам (кафе, рестораны, столовые, кофейни и т.д.)
   - Изучение географического распределения по районам Москвы
   - Выявление зависимости между типом заведения и средним чеком
   - Анализ рейтингов и отзывов

3. **Визуализация результатов**
   - Создание графиков распределения по категориям
   - Визуализация ключевых трендов

4. **Формулирование рекомендаций**
   - Выявление свободных ниш на рынке
   - Определение оптимального района для открытия
   - Рекомендации по типу заведения и ценовой политике

5. **Выводы**
   - Обобщение результатов анализа
   - Практические рекомендации для бизнеса

**Стек:** Python, Pandas, Matplotlib, Seaborn, Phik, Jupyter Notebook
