# practicum_projects
Мои учебные проекты с курса Яндекс.Практикум по анализу данных.

## Список проектов

### 1. Анализ рынка общественного питания Москвы
[Подробнее](Moscow_restaurants_analysis/README.md) [Открыть тетрадку ➜](Moscow_restaurants_analysis/moscow_restaurants_analysis.ipynb)

**Задачи:** анализ структуры рынка, визуализация, выводы и рекомендации для открытия заведения

**Стек:** Python, Pandas, Matplotlib, Seaborn, Phik, Jupyter Notebook

**Краткие итоги:**
- Топ-3 формата: кафе, рестораны, кофейни
- ЦАО — лидер по концентрации (26.7%) и ценам (1 191 ₽)
- Рекомендация: ресторан/бар в ЦАО/ЗАО/САО с ценами выше среднего

### 2. Развитие игровой индустрии за 2000–2013 гг.
[Подробнее ➜](Videogames_analysis/README.md) [Открыть тетрадку ➜](Videogames_analysis/videogames_analysis.ipynb)

**Задачи:** анализ платформ, категоризация оценок пользователей и критиков, выявление трендов

**Стек:** Python, Pandas, Jupyter Notebook

**Краткие итоги:**
- Топ-3 платформы: PS2 (2 127), DS (2 120), Wii (1 275)
- Большинство игр получают средние оценки (Количество оценок пользователей: 4 081, критиков: 5 422)
- Хороших игр значительно больше, чем плохих
