# Анализ рынка общественного питания Москвы

## Описание проекта
Исследование рынка заведений общественного питания Москвы на основе открытых данных. 
Цель - выявить текущие тренды, определить перспективные ниши и сформулировать рекомендации заказчику для открытия нового заведения.

## Задачи
1. **Сбор и предобработка данных**
   - Очистка данных от пропусков и выбросов
   - Приведение данных к единому формату
   - Обогащение данных (добавление признаков: режим работы, ценовой сегмент)

2. **Исследовательский анализ данных (EDA)**
   - Анализ структуры заведений по типам (кафе, рестораны, столовые, кофейни и т.д.)
   - Изучение географического распределения по районам Москвы
   - Выявление зависимости между типом заведения и средним чеком
   - Анализ рейтингов и отзывов

3. **Визуализация результатов**
   - Создание графиков распределения по категориям
   - Визуализация ключевых трендов

4. **Формулирование рекомендаций**
   - Выявление свободных ниш на рынке
   - Определение оптимального района для открытия
   - Рекомендации по типу заведения и ценовой политике

5. **Выводы**
   - Обобщение результатов анализа
   - Практические рекомендации для бизнеса

## Стек 
Python, Pandas, Matplotlib, Seaborn, Phik, Jupyter Notebook

## Результаты
- Выявлены самые популярные типы заведений в Москве
- Определены районы с наибольшей концентрацией заведений
- Обнаружены корреляции между типом заведения и средним чеком
- На основе анализа предложены рекомендации по открытию нового заведения

## Ключевые выводы

- **Самые популярные форматы:** кафе (2 376), рестораны (2 042), кофейни (1 413)
- **ЦАО** - лидер по концентрации заведений (26.7% от всех)
- **62%** заведений - несетевые, 38% - сетевые
- **Цены выше в центре:** средний чек в ЦАО - 1 191 ₽, в ЮВАО - 654 ₽
- **Рейтинг зависит от цен:** чем дороже, тем выше оценка (от 4.17 до 4.44)
- **Самый высокий рейтинг** - у баров/пабов (4.39)
- **Топ сетей:** Шоколадница (119 точек), Додо Пицца (лучший рейтинг - 4.29)

## Рекомендации
- **Формат:** ресторан или бар/паб
- **Цены:** выше среднего / высокие
- **Локация:** ЦАО, ЗАО или САО
- **Избегать:** ЮАО (низкий чек + высокая конкуренция)
